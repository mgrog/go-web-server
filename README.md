# Go (Gin) Server

This is an example of a Gin based web server implementing common web use cases

## Implemented Features

- [x] Setup basic server
- [x] REST routes
- [x] Database connections
- [x] Database migration tooling with sql-migrate
- [x] Custom middleware
- [x] Setup GraphQL endpoints
- [x] Dataloaders for N+1 queries
- [x] SQL queries with sqlx
- [ ] Error handling
- [ ] Testing
- [ ] Add logging
- [ ] Authentication
- [ ] Swagger documentation
- [ ] Dockerized deployment
